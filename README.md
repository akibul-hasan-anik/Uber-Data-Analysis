# Uber Data Analysis
This project performs data analysis and machine learning on an Uber ride dataset to explore patterns in ride demand and predict ride-related outcomes.

# Features
* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Feature engineering
* Machine learning model training
* Ride demand prediction using Random Forest

# Technologies Used
* Python
* Pandas
* NumPy
* Scikit-learn
* Jupyter Notebook

# Goal
The goal of this project is to analyze Uber ride data and build a machine learning model that can help predict ride demand based on historical data.
